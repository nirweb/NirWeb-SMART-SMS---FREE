افزونه پیامک ووکامرس و ورودپرس | با این افزونه میتوانید انواع اطلاع رسانی های پیامکی برای ووکامرس و وردپرس خود داشته باشید.

== Description ==
**NirWeb SMART SMS**  افزونه پیامک ووکامرس و وردپرس نیر وب یک افزونه  کاملا کاربردی است که تمام اطلاع رسانی های مورد نیاز شما در ووکامرس را برای شما فراهم میکند.

= امکانات =
* ارسال پیامک به مشتری و مدیرکل و مدیر محصول (فروشندگان) در صورت ثبت و تغییر وضعیت سفارش
* امکان تنظیم ارسال پیامک در وضعیت های مختلف سفارش (تکمیل شده، معلق و...)
* قابلیت تغییر متن و شخصی سازی هر پیامک برای مدیر، مشتری و فروشنده برای هر وضعیت سفارش
* سیستم خبرنامه حرفه ای محصولات برای با خبر سازی کاربران از وضعیت موجود شدن محصولات، فروش ویژه شدن و ....
* و سایر ویژگی های حرفه ای دیگر

= درگاه های پیامک =
* IPPANEL ( مانند فراز اس ام اس  )
* MeliPayamak.com
* sms.ir 

= پشتیبانی =
*  برای دریافت پشتیبانی لطفا به سایت نیر وب مراجعه کنید

== Installation ==
1. Upload `nirweb-smart-sms-free` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Complete NirWeb SMART SMS free Settings
